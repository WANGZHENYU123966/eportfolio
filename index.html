SDG 03: Good Health and Well-being

This portfolio is aligned with SDG 3: Good Health and Well-being. I chose this SDG because mobile applications can help people build positive daily habits, monitor their progress, and improve their physical and mental well-being through simple and accessible technology.

Labs

These five labs show my progress from building basic user interfaces to developing Android applications with navigation and local database storage.

Lab 1 · UI

Building a Basic User Interface

This lab focused on creating the foundation of a mobile interface using Jetpack Compose.

Through Lab 1, I learned how to create a basic mobile user interface using Jetpack Compose. This lab helped me understand the importance of layout structure, spacing, text arrangement, and visual hierarchy. It also became the foundation for designing more complete and user-friendly screens in later labs.

Lab 2 · Interaction

User Input and Interaction

This lab introduced interactive components and user-driven state changes.

In Lab 2, I learned how to make an application respond to user actions. I practiced using buttons, text fields, state changes, and event handling to create a more interactive user experience. This lab helped me understand that a good mobile app should not only look clear, but also respond smoothly and meaningfully to users.

Lab 3 · Material Design

Modern Android Design

This lab improved the visual quality and consistency of the app interface.

Lab 3 introduced me to Material Design and helped me improve the visual quality of my application. I learned how to use components such as cards, buttons, colors, and icons to make the interface more modern and consistent. This lab showed me that design is an important part of usability, not just decoration.

Lab 4 · Navigation

Multi-Screen Navigation

This lab focused on connecting different screens in a logical user flow.

In Lab 4, I learned how to connect multiple screens using navigation. This was an important step because real mobile applications usually contain many pages and features. By practicing navigation, I understood how to create a logical user flow and make the app easier for users to explore.

Lab 5 · Room

Local Data Storage

This lab introduced Room Database and data persistence in Android development.

Lab 5 helped me understand how to store and manage local data using Room Database. I learned how entities, DAO, database classes, and ViewModel work together to save and retrieve information. This lab was very useful because it showed me how an application can keep user data even after it is closed.
